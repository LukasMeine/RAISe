get()

get(string  $item, array  $templateEngine = array()) : \UIoT\Interfaces\SettingsInterface|array<mixed,\UIoT\Interfaces\SettingsInterface>

This method returns an item by his identifier.

Normally the Factory will receive the string, and route (do) the search in his Data with a specific rule from the Factory.

The templateEngine variable is useful when the Router need optional parameters. In the case of MessageFactory, the templateEngine parameter is used to populate variables into values passed by parameter.

Necessary the return need be an object or array of objects.
